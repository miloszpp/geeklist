GeekList

Task management app with a command line.
Created by Miłosz Piechocki using Xamarin Studio

Task creation

You can just type the task description straight away to the command box. You can use ^ and # special characters to quickly specify the due date or priority.

visit Agnes ^today #3 do the dishes ^tomorrow #1 do the laundry ^Jul29

Task manipulation

You can use the following commands for task manipulation:

  • rm - remove the task
  • pp - postpone the task
  • do - complete the task

All of these tasks take one argument - the task id. Task id is displayed in the square brackets, next to the task description.

do a.b pp b.a

It is possible to easily manipulate whole groups of tasks. Instead of specifying the task id just provide the section id. If, for example, you want to clear your day, it is as simple as:

pp a

Command names are not case sensitive.
